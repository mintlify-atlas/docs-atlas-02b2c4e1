## Prerequisites

The build requires clang++ (C++23), clang, and several system libraries. On Ubuntu/Debian:

```bash
sudo apt install \
    clang \
    libsdl2-dev \
    libsdl2-image-dev \
    libsdl2-ttf-dev \
    libgl-dev \
    libglm-dev
```

<Note>
GLAD and tinyobjloader are bundled in the `headers/` directory and do not need to be installed separately.
</Note>

## Build steps

<Steps>

<Step title="Clone or unpack the source">

Make sure the project directory contains `Makefile`, `src/`, `headers/`, `shaders/`, and `assets/`.

</Step>

<Step title="Build the binary">

Run `make` from the project root. Clang++ compiles all `.cpp` files in `src/` and clang compiles `src/glad.c`. Object files are placed in `build/`.

```bash
make
```

On success the output binary is `./game`.

</Step>

<Step title="Run the program">

```bash
make run
```

This builds (if necessary) and then executes `./game`.

</Step>

</Steps>

## Makefile targets

| Target | Description |
| --- | --- |
| `make` / `make all` | Compile everything and link `./game` |
| `make run` | Build and run `./game` |
| `make clean` | Remove the `build/` directory and the `game` binary |

To clean and do a full rebuild:

```bash
make clean && make
```

## Compiler and flags

The Makefile uses:

- **Compiler:** `clang++` for C++ sources, `clang` for `glad.c`
- **C++ standard:** `-std=c++23` (required — the code uses `<print>` and other C++23 features)
- **Warning flags:** `-Wall -Wextra`
- **Debug info:** `-g`

## Directory layout

```
project/
├── Makefile
├── src/          # C++ and C source files
│   ├── main.cpp
│   ├── createWindow.cpp
│   ├── objrender.cpp
│   ├── phy1.cpp
│   └── glad.c
├── headers/      # Public headers and bundled dependencies
│   ├── createWindow.h
│   ├── shapes.h
│   ├── obj.h
│   ├── phy.h
│   ├── glslread.h
│   ├── glad/     # GLAD OpenGL loader
│   ├── KHR/      # Khronos platform headers
│   └── tinyobj/  # tinyobjloader
├── shaders/      # GLSL vertex and fragment shaders
├── assets/       # OBJ models and textures
└── build/        # Compiled object files (generated by make)
```
